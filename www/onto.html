Web-VTT as linked data ontology

This is a quick-and-dirty HTML representation of an ontology representing Web-VTT concepts as linked data.

Classes

VideoAnnotationDataset

Instances of this class are RDF datasets representing the information conveyed by Web-VTT files.

Cue

Instances of this class are Cues as defined by the Web-VTT specification. Cues with an identifier have an IRI based on that identifier, while cues without any identifiers are represented by blank nodes.

Properties

hasCue

This property links a video annotation dataset to all the cues it defines.

describesFragment

This property links an instance of cue to the media fragment it describes.

annotatedBy

This property links a Media fragment to the payload of a cue describing it.
